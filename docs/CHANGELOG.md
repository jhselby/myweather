## v4.78 • 2026-04-21
* **Sea Breeze Card — Fixed False "No Data" Error**
  * Root cause: renderer guard was `if (!sb.likelihood)`, which treats `likelihood === 0` as falsy and short-circuits to "No sea breeze data available"
